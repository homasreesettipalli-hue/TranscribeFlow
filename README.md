# TranscribeFlow – AI Audio Transcription System

## Overview
TranscribeFlow is an AI-powered audio transcription application that converts speech from audio files into text. It helps users generate transcripts for meetings, lectures, interviews, and other audio recordings quickly and efficiently.

## Features
- Audio-to-text transcription
- Speech recognition
- Easy-to-use interface
- Fast transcript generation
- Support for audio file uploads
- Download and review transcripts

## Technologies Used
- Python
- HTML
- CSS
- JavaScript
- Flask
- Speech Recognition Libraries

## Installation

1. Clone the repository

```bash
git clone https://github.com/homasreesettipalli-hue/TranscribeFlow.git
```

2. Navigate to the project folder

```bash
cd TranscribeFlow
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

## Usage

1. Launch the application.
2. Upload an audio file.
3. Wait for processing.
4. View the generated transcript.
5. Download or copy the transcript.
